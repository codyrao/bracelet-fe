# DIY 手链 H5 原型

一个无需构建或安装依赖的 DIY 手链演示页面，优先适配手机端，也支持电脑端自适应双栏编辑。

## 使用

直接用现代浏览器打开 `index.html`。页面使用本地模拟珠子数据，珠子图片来自 DIYrock 的珠子目录配方生成的本地透明圆珠 PNG。

## 功能

- 分类和搜索珠子，并可在「DIY 手链 / 设计广场」之间切换。
- 设计广场内置设计师款和优秀客订各 8 个按点赞数排序的示例设计，点击卡片可查看详情、直接生成确认订单，或复制整条手链回 DIY 页面继续二次设计。
- DIY 页面支持把当前手链保存到「我的设计」，先使用浏览器本地缓存持久化，最多保存 20 个；我的设计可点击回到 DIY 继续修改，也可删除。
- 点击商品卡或把商品卡拖到手链舞台加入手链；轻点手链珠子播放滚动动效并选中，长按删除；可一键清空。
- 珠子达到 20 颗时不能继续添加，会弹窗提示；仍可把商品卡拖到已有珠子上替换，位置和总数保持不变。
- 通过「3D 展示」按钮查看整条手链的地球仪式斜向环绕 canvas 3D 光影预览，展示实心珠子、珠孔、珠间麻绳弧线、珠子遮挡和材质高光；把商品卡拖到该按钮上可临时预览加入后的整圈效果。
- 首页和 3D 展示弹窗都支持打开「确认订单」弹窗，展示手链平面预览图、珠子数量、同款珠子的数量 × 单价、小计和总价；该确认只作为前端演示，不会提交真实订单。
- 实时显示已选数量、20 颗满珠提示、估算手围和总价。
- 手机端使用纵向全屏编辑流程并关闭双击放大，电脑端在 900px 以上自动切换为左侧手链预览、右侧珠子面板的双栏 UI。
- 16 颗后珠子会基本覆盖串线，最多可加到 20 颗；达标过程中手链半径保持稳定，避免突然缩小。
- 使用低开销的入场、按压、呼吸、弹性飞入与悬停微动效；点击添加珠子时会按移动距离自适应过渡时长，并遵循系统“减少动态效果”偏好。
- 支持可关闭的轻量交互音效，音效由浏览器 WebAudio 实时生成，不依赖外部音频素材。
- 手链按顶视圆环排布，串线使用柔和绳线质感，珠子采用多层高光、晶体纹理、折射与阴影模拟材质。

## 发布到 GitHub Pages

这个项目是纯静态页面，适合直接用 GitHub Pages 免费发布。

1. 在 GitHub 新建一个公开仓库，例如 `hand-diy-demo`。
2. 把本目录文件推送到仓库的 `main` 分支。
3. 进入仓库 `Settings` → `Pages`。
4. `Source` 选择 `Deploy from a branch`，`Branch` 选择 `main`，目录选择 `/(root)`。
5. 等待部署完成后访问 `https://你的用户名.github.io/hand-diy-demo/`。

## 图片来源

演示珠子目录来自 MIT license 的 [air158/DIYrock](https://github.com/air158/DIYrock) catalog。该仓库没有独立单珠位图，本项目按其颜色、名称、尺寸和价格配方生成 `assets/beads/diyrock-*.png` 本地透明圆珠图；`assets/sources/diyrock-catalog.json` 保留来源和许可信息，便于追溯和替换。
