# DIY 手链 H5 原型

一个无需构建或安装依赖的移动端 DIY 手链演示页面。

## 使用

直接用现代浏览器打开 `index.html`。页面使用本地模拟珠子数据，不会发起网络请求。

## 功能

- 分类和搜索珠子。
- 点击商品卡加入手链；点击手链珠子删除；可一键清空。
- 实时显示已选数量、最少 16 颗的提示和总价。
- 在 375px 至 430px 宽度下适配移动端布局。

## 测试

```powershell
node --test tests/app.test.mjs
```

## 范围

这是前端原型，不包含商品接口、库存、订单、支付、登录、持久化、分享、拖拽或 3D 预览。
