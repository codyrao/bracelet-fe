# DIY 手链 H5 原型

一个无需构建或安装依赖的移动端 DIY 手链演示页面。

## 使用

直接用现代浏览器打开 `index.html`。页面使用本地模拟珠子数据，并从 Wikimedia Commons 加载演示珠子图片。

## 功能

- 分类和搜索珠子。
- 点击商品卡或把商品卡拖到手链舞台加入手链；轻点手链珠子播放滚动动效并选中，长按删除；可一键清空。
- 珠子达到 16 颗时不能继续添加，会弹窗提示；仍可把商品卡拖到已有珠子上替换，位置和总数保持不变。
- 通过「3D 展示」按钮查看整条手链的地球仪式斜向环绕 canvas 3D 光影预览，包含真实串线、珠子遮挡、投影和材质高光；把商品卡拖到该按钮上可临时预览加入后的整圈效果。
- 实时显示已选数量、16 颗满珠提示和总价。
- 在 375px 至 430px 宽度下适配移动端布局。
- 使用低开销的入场、按压与悬停微动效，并遵循系统“减少动态效果”偏好。
- 手链按顶视圆环排布，串线使用柔和绳线质感，珠子采用多层高光、晶体纹理、折射与阴影模拟材质。

## 测试

```powershell
node --test tests/app.test.mjs
```

## 发布到 GitHub Pages

这个项目是纯静态页面，适合直接用 GitHub Pages 免费发布。

1. 在 GitHub 新建一个公开仓库，例如 `hand-diy-demo`。
2. 把本目录文件推送到仓库的 `main` 分支。
3. 进入仓库 `Settings` → `Pages`。
4. `Source` 选择 `Deploy from a branch`，`Branch` 选择 `main`，目录选择 `/(root)`。
5. 等待部署完成后访问 `https://你的用户名.github.io/hand-diy-demo/`。

## 图片来源

演示商品图来自 Wikimedia Commons 的开放图片页面，实际商品上线时建议替换为自有商品图。每个商品在 `app.js` 里保留 `sourceUrl`，便于追溯和替换。

## 范围

这是前端原型，不包含商品接口、库存、订单、支付、登录、持久化、分享或真实 WebGL 3D 模型。
