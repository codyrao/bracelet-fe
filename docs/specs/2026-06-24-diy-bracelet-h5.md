# DIY 手链 H5 原型规格

## 目的

在空仓库中交付一个可在本地浏览器直接打开的移动端 DIY 手链 H5 原型。用户可从珠子商品库筛选并点击添加珠子，预览手链、查看已选数量与总价，并可删除或清空已选珠子。

## 改动分支

feature/diy-bracelet-h5

## 实现方式

1. 创建单页移动端界面 -> 还原已提供设计稿的顶部栏、手链舞台、状态栏和底部珠子抽屉。
2. 提供本地模拟珠子商品数据 -> 支持分类、搜索、规格与价格展示，不接入远程接口。
3. 提供手链选择状态 -> 支持点击商品添加、点击已选珠子删除、清空选择，以及实时总价和数量提示。
4. 提供响应式视觉排布 -> 根据已选珠子按椭圆轨道重排，空状态显示引导文案。

## 架构设计

页面为静态 HTML、CSS 和 JavaScript，不引入框架、构建工具或第三方依赖。商品数据与当前选择状态保存在浏览器内存中；手链轨道使用 SVG，珠子使用可聚焦的 HTML 按钮叠加渲染。

## 性能优化

商品数据规模仅覆盖演示所需的少量本地数据。仅在选择、删除、分类或搜索条件变化时重绘商品列表和手链；不使用动画循环、网络请求或大型图形依赖。

## 安全控制

不收集、存储或发送用户数据；不加载第三方脚本；商品名称与搜索输入以文本方式渲染，不拼接为 HTML。

## 已确认边界

- 首版为可本地打开的静态 H5 原型。
- 使用模拟商品数据，不接商品、库存、订单、支付或用户接口。
- 包含分类、搜索、添加珠子、删除珠子、清空、数量提示和总价计算。
- 不包含自由拖拽、3D 预览、保存作品、分享、登录或下单。
- 以已提供的移动端设计稿为主要视觉参考。

## 实现效果

- 空手链显示灰色椭圆轨道与选择引导。
- 点击商品卡后，珠子显示在手链轨道上，已选数量和总价同步更新。
- 点击手链中的珠子可删除该珠子；“清空”可恢复空状态。
- 左侧分类和搜索框可共同筛选商品卡。
- 页面适配 375px 至 430px 宽度的移动端视口，并保留键盘可见焦点。

## 兼容性

仓库没有既有功能、依赖或初始提交。本功能为新建静态页面，不影响外部系统；面向现代桌面和移动浏览器。

## 风险

- 当前仅有一张设计截图，图标、商品实拍图和精确字号只能用近似本地视觉元素代替。
- 最小珠子数量、目标手链周长等商业规则尚未定义，原型仅展示可调整的演示规则。
- 未接后端时，刷新页面会丢失当前 DIY 选择。
